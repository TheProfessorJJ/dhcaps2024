## Rules for Landing Pages

The product grid on all country landing pages shows **8 products**. Always use this exact set:

```
1. Snapback Hats
2. Classic Trucker Mesh
3. 100% Cotton Sports Cap        ← name must include "100%"
4. Pigment Dyed Washed Hats      ← name must include "Hats"
5. Polyester Hats With Sublimation Print  ← full name, not just "Sublimation Print"
6. Mesh Trucker Hats With Side Tape       ← full name
7. Promotional Caps
8. Size Fitted Hats
```

Never use `https://www.dhcaps.com/wp-content/uploads/...` URLs for these 8 product cards — those old WP upload filenames do not match the product types correctly.
